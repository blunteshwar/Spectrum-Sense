sp-popover

The Popover component displays floating content relative to a trigger element.

Usage

When opening a popover on pointerdown, ensure to set the placement prop correctly.

Basic Example


<sp-popover placement="bottom">
    <sp-button slot="trigger">Open Popover</sp-button>
    <p>Popover content</p>
</sp-popover>

Pointer Events

To open on pointerdown, use the open attribute with an event listener:


const popover = document.querySelector('sp-popover');
const button = document.querySelector('sp-button');
button.addEventListener('pointerdown', () => {
    popover.open = true;
});
